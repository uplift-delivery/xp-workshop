# Sudoku

## Rules

1. Grid shall be 9x9
2. The grid shall contain 9 - 3x3 grids
3. Players must place the numbers 1-9 in each:
   1. Column
   2. Row
   3. 3x3 Grid
4. Players may not repeat any single number in any:
   1. Column
   2. Row
   3. 3x3 Grid
